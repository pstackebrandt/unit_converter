Description

Examples are great, but they are not really helpful. Let's take a step further and create a program that converts kilometers to meters. Remember that one kilometer equals exactly 1000 meters.

The program should prompt the user to enter the number of kilometers with the following message: Enter a number of kilometers: . The program should then output the equivalent of x kilometers in meters in the following format: x kilometers is y meters.

Objectives

  1. The program should convert the given number of kilometers to meters.
  2. The program should ask the user only once, not infinitely.

Examples

Example 1

Enter a number of kilometers: 25
25 kilometers is 25000 meters

Example 2

Enter a number of kilometers: 12
12 kilometers is 12000 meters
